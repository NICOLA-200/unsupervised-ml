## Overview of Machine Learning Techniques

This project demonstrates unsupervised learning and recommender systems using Python, Scikit-learn, and Jupyter Notebook.


### Unsupervised Learning

Discover patterns in unlabeled data:
Clustering: K-Means, DBSCAN, Hierarchical Clustering
Dimensionality Reduction & Manifold Learning: PCA, t-SNE

### Project 1: Unsupervised Learning

Explore patterns in movie ratings using unsupervised learning algorithms:
Clustering: K-Means, DBSCAN, Hierarchical Clustering
Dimensionality Reduction & Manifold Learning: PCA, t-SNE
Tools: Python, Scikit-learn, Jupyter Notebook

### Project 2: Recommender Systems

Build personalized movie recommendations:
Suggest movies based on user preferences
Demonstrate collaborative filtering and similarity-based approaches
Tools: Python, Scikit-learn, Jupyter Notebook
